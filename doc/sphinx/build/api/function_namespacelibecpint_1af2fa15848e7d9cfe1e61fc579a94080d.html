TwoIndex<double> libecpint::realSphericalHarmonics(int lmax, double x, double phi)¶

Calculates real spherical harmonics S_lm(theta, phi) for all l, m up to lmax

Return

a matrix S(l, l+m) of the spherical harmonic values

Parameters

  • lmax: - the maximum angular momentum needed

  • x: - cos(theta), where theta is the polar angle in spherical coordinates

  • phi: - the azimuth angle in spherical coordinates
