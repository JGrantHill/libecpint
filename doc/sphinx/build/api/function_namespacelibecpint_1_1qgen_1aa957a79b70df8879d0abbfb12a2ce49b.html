Function Documentation¶

void libecpint::qgen::rolled_up(int lam, int LA, int LB, ThreeIndex<double> &radials, FiveIndex<double> &CA, FiveIndex<double> &CB, TwoIndex<double> &SA, TwoIndex<double> &SB, AngularIntegral &angint, ThreeIndex<double> &values)¶

For integral functions that have not been unrolled, this loops over the angular and radial integrals to provide the overall ECP integrals, returning in an array indexed as (shellA, shellB, mu), for mu = -lam .. lam

Parameters

  • lam: - angular momentum of ECP shell

  • LA: - angular momentum of basis shellA

  • LB: - angular momentum of basis shellB

  • radials: - array of radial integrals, indexed as (N, lA, lB)

  • CA: - binomial expansion coefficients of shellA

  • CB: - binomial expansion coefficients of shellB

  • SA: - spherical harmonics of shellA

  • SB: - spherical harmonics of shellB

  • angints: - object containing the angular integrals

  • values: - the array in which the results are returned
